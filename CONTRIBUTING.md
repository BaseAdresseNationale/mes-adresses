### Comment contribuer à Mes-Adresses?

Avant tout, merci de votre intérêt pour le projet Mes-Adresses.

Vous souhaitez une nouvelle fonctionnalité ou vous avez constaté un bug en utilisant l'application? Le moyen le plus simple est de nous contacter directement via Matermost sur le channel #mes-adresses ou de créer une issue sur Github sur ce dépôt.

Si vous souhaitez contribuer au code du projet, c'est également possible mais nous vous recommandons de prendre d'abord contact avec nous via les canaux mentionnés plus haut.
